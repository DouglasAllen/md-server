Sinatra

Sinatra is a DSL for quickly creating web applications in Ruby with minimal effort:

# myapp.rb
require 'sinatra'

get '/' do
  'Hello world!'
end

Install the gem:

gem install sinatra

And run with:

ruby myapp.rb

View at: http://localhost:4567

It is recommended to also run gem install thin, which Sinatra will pick up if available.
